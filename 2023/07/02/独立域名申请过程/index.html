建立 github 博客

大概用了一个月，确信学会了用 hexo 布署和更新 github 博客。在这期间，更换主题，增加评论等功能，也一一学会。会者不难，但没学会之前，觉得难度很大。现在看来这难度之大，多半是畏难情绪造成的。

绑定独立域名

这一步很难，即来自心理也来自技术。

要绑定独立域名，首先得购买独立域名，其次还要 dns 解析，域名是国内的话，还要备案。中间等待多久，完全属于未知情况。

看了很多个 B 站教程，看了很多知乎贴子。由于自身存在畏难情绪，很多教程看完就关了。因此兜兜转转，费了很长时间。

今天上午，顺利完成。在 NameSilo 上注册账户，购买域名（支持支付宝，可以自动续费），卡在了 DNS 域名解析上。网络上有很多解决办法，最重要的是等待时间。试了半天，一直没解析成功。到了饭点，起身做饭，回来一刷屏发现已经成功。

总结：网络教程多数是有效的，关键有两点：1、要耐着性子看下去，给点时间让其缓冲；2、要克服畏难情绪，勇于尝试，只要有敢试着做下去的心态，事情就有成功的可能。

后续

一

如果只是在 github page 里的 custom domain 里填写独立域名的话，每次写完博客 hexo deploy 后，网页就打不开了，原因 custom domain 也随之更新清空了。

解决办法：在 hexo 源文件下的 source 目录下，创建 CNAME 的文件（注意全部大写），里面写上独立域名。经检测，这样是成功的。
