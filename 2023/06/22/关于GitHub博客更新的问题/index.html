我有多台电脑，我的诉求是在不同电脑上可以及时更新我的github博客。

办公室有一台 win10，家里有两台 win10，一台 mac，并且已经通过 onedrive 实现文件实时同步。这是我想在不同电脑上更新 github 博客的大前提。

网上有很多不同电脑更新博客的教程。多数是通过在 github 仓库上另建分支来解决的，这种解决办法在我看来过于复杂，因此一直寻找更简单的解决办法。经过无数次摸索，最终实验成功。

方法如下：

  1. 在新电脑上创建ssh；
  2. 将ssh粘贴的GitHub里面；
  3. 把博客本地文件夹全部同步到新电脑上；
  4. 就像在第一台电脑上一样用hexo命令写博客

经过mac和另一台win10的测试，更新全部成功。
